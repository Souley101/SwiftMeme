# SwiftMeme v0.2.0

## Synopsis

SwiftMeme is a source discovery and keyword monitoring tool for tracking memes online.

## Requirements

* [Python 2.7.1+](http://www.python.org/download/releases/2.7.1/)
* [Flask 0.6.1+](http://flask.pocoo.org/)
